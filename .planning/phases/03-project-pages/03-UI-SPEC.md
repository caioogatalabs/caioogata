| Role | Token | Usage in Phase 3 |

## Interaction Contract

### Hero Section
- **Entrance:** `-entrance -slide-up -a-0` on title, `-entrance -fade -a-1` on metadata, `-entrance -scale-in -a-2` on hero image(s). Gated behind `html.-loaded.-ready`.
- **Hero image:** Full-width (`12-col`) or split (`6-6`). 12px border radius. `overflow-hidden` on container. No hover state on hero images.
- **Data-stamp:** `PRJ_{year} // {index}` in Cascadia Mono 11px uppercase `tracking-[0.88px]` -- matches ProjectCard pattern exactly.

### Gallery 1 -- Staggered Grid
- **Image reveal:** Each image uses `useScrollReveal` -- `clip-path: inset()` with `round 12px` expands as image scrolls into view. `startFraction: 0.85`, `endFraction: 0.3` (same defaults as ProjectCard).
- **Row gap:** 24px between rows on desktop, 16px tablet, 8px mobile.
- **Image radius:** 12px (`--radius-component-md`).
- **No hover effect on images.** Images are display-only (D-08: no lightbox).

### Gallery 2 -- Feature List
- **Layout:** 3-6-3 grid. Feature name (left), image (center), description (right).
- **Image reveal:** `useScrollReveal` on each image container.
- **Parallax float:** `useScrollParallax` with `factor: 0.1`. Image container has `overflow: hidden`, inner image applies `translateY` from hook. Maximum drift: approx 20px in either direction.
- **Mobile stack:** All columns become `mobileSpan={4}` (full width). Order: feature name, image, description -- vertically stacked.
- **Reduced motion:** Parallax disabled. Image appears static at center position.

### Gallery 3 -- Full Detail
- **Layout:** 4-8 grid (text left, image right) or 8-4 (image left, text right). Per-block configurable.
- **Image:** Minimum height 100vh. `object-cover` on image. 12px radius.
- **Text:** Body text with heading. Vertically centered relative to image.
- **Entrance:** `-entrance -slide-up` triggered by `-inview` on the containing section.

### Inter-Project Navigation
- **Position:** Rendered below hero (sticky at top on scroll) AND at bottom of content (static).
- **Layout:** `flex justify-between`, full-width with page padding (`px-5 md:px-8 lg:px-16`).
- **Prev link (left):** `<- {previous project title}` in Cascadia Mono 14px. If no prev, show `<- Back to Home`.
- **Next link (right):** `{next project title} ->` in Cascadia Mono 14px. If no next, show `Back to Home ->`.
- **Hover state:** `text-text-secondary` default -> `text-text-primary` on hover. Transition: `transition-colors duration-300`.
- **Keyboard:** Left arrow = prev, Right arrow = next, Escape = back to Home. Focus ring uses `border-focus` token.
- **Top border:** `border-t border-border-primary` on both instances.
- **Sticky behavior (top nav):** `position: sticky; top: 0; z-index: 40` with `bg-bg` background + `backdrop-blur-xl` for glassmorphism effect. Below StickyHeader z-index (z-50).

### Info Block
- **Background:** `bg-bg-surface-secondary` -- tonal stacking, no border (Design.md "No-Line Rule").
- **Layout:** 12-col grid with 5 internal columns: Project (3-col), Client/Role (2-col), Technologies (3-col), Year (2-col), Links (2-col).
- **Column headers:** Cascadia Mono 12px uppercase `tracking-wider text-text-tertiary`.
- **Column values:** Fabio XM 16px `text-text-primary`.
- **Links:** Underline on hover, `text-text-brand` color.
- **Padding:** 32px vertical, page horizontal padding.
- **Mobile:** Stacks into 2-column grid (label + value pairs).

### Credits
- **Layout:** Simple list below info block. Each credit: `{name} - {role}` with LinkedIn link if available.
- **Name link:** `text-text-secondary` -> `text-text-primary` on hover, underline on hover.
- **Spacing:** 8px between credit items, 48px above credits section.

### Page-Level
- **Page padding:** `px-5 md:px-8 lg:px-16` (64px desktop) -- matches all Home sections.
- **Section vertical spacing:** 64px between major blocks (hero -> challenge, challenge -> gallery, gallery -> info block).
- **Loading gate:** Entrance animations gated behind `html.-loaded.-ready` (same as Home).
- **Scroll:** Native scroll only. No smooth-scroll libraries.

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Skip link | Reuse existing `SkipLink` from Home. Target `#main-content` on project page. |
| Keyboard navigation | Left/Right arrows for prev/next project. Escape returns to Home. Tab moves through interactive elements. |
| Focus indicators | `outline: 2px solid var(--color-border-focus)` offset 2px on all focusable elements. |
| Image alt text | All gallery images use `title` from data model. Decorative images use `alt=""`. |
| Navigation landmarks | `<nav aria-label="Project navigation">` on both nav bars. `<main id="main-content">` wrapper. |
| Reduced motion | `prefers-reduced-motion: reduce` disables all entrance animations, scroll-reveal (images show fully), parallax (static position). |
| Color contrast | All text meets WCAG AA. `text-tertiary` used only for text 18px+ (noted in semantic.css). |
