# caioogata

Portfolio V1 - LLM First

## Sobre

Portfólio pessoal desenvolvido com foco em consumo por LLMs (Large Language Models). Otimizado para ser facilmente processado e compreendido por assistentes de IA.

## Tecnologias

- **Next.js 15** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização utilitária
- **React Hot Toast** - Notificações

## Características

- ✅ Design otimizado para consumo por LLM
- ✅ Suporte a múltiplos idiomas (PT-BR / EN)
- ✅ Copy as Markdown para assistentes de IA
- ✅ Acessibilidade (ARIA, Skip Links)
- ✅ Responsivo e fluido
- ✅ Export estático (output: 'export')

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Iniciar servidor de produção
npm start
```

## Estrutura

```
src/
├── app/              # App Router (Next.js 15)
├── components/       # Componentes React
│   ├── hero/        # Seção Hero
│   ├── layout/      # Layout (Footer)
│   ├── sections/    # Seções do portfólio
│   └── ui/          # Componentes UI reutilizáveis
├── content/         # Conteúdo multilíngue (JSON)
└── lib/             # Utilitários
```
